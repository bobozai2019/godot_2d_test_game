# 阶段 1 基础开发计划

## 目标

按 `doc/2026-04-11-2d阴阳师游戏计划.md` 启动阶段 1，搭建可继续扩展的 Godot 2D demo 基础结构。

## 步骤

1. 创建 `scenes/`、`scripts/`、`assets/`、`ui/` 目录。
2. 新建 `main.tscn`、`player.tscn`、`enemy.tscn`、`hud.tscn`，并在主场景内放置小型测试竞技场、玩家、敌人和 HUD 实例。
3. 配置 `project.godot` 主场景为 `res://scenes/main.tscn`。
4. 添加输入动作：`move_up`、`move_down`、`move_left`、`move_right`、`attack`、`skill_1`、`skill_2`、`interact`、`pause`。
5. 运行 `godot --headless --path . --quit` 做基础加载检查，并查看 Git 变更。
