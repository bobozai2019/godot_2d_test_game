# 阶段 2 战斗原型开发计划

## 目标

在阶段 1 场景基础上实现最小可玩的战斗原型：玩家移动、剑士占位动画、史莱姆敌人追击/攻击/死亡、波次生成、生命值与冷却数据。

## 步骤

1. 新增共享数据脚本 `combat_stats.gd` 和简易图集动画脚本 `sheet_animator.gd`。
2. 为 `player.tscn` 添加移动、攻击、受伤、死亡逻辑和剑士图集动画。
3. 为 `enemy.tscn` 添加待机、追击、攻击、受伤、死亡逻辑和史莱姆图集动画。
4. 为 `main.tscn` 添加敌人生成器、波次胜利条件和 HUD 数据同步。
5. 为 `hud.tscn` 添加脚本接口显示生命值、技能冷却、波次与胜负状态。
6. 运行 Godot headless 加载检查，并查看 Git 变更。
